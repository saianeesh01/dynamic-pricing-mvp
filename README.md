# AI-Driven Dynamic Pricing Platform

## Overview
Phase 1 MVP of the Dynamic Pricing Platform.
A Flask-based pricing engine with an Angular dashboard.

## Tech Stack
- **Backend**: Python (Flask), SQLAlchemy, SQLite
- **Frontend**: Angular, TailwindCSS
- **Infrastructure**: Local Development

## Setup

### Backend
1. Navigate to `backend/`.
2. Create virtual environment: `python -m venv venv`
3. Activate: `.\venv\Scripts\Activate` (Windows)
4. Install: `pip install -r requirements.txt`
5. Run: `python app.py`

### Frontend
1. Navigate to `frontend/`.
2. Install: `npm install`
3. Run: `ng serve`

## Features
- Real-time demand pricing (Simulated).
- Rule-based adjustments (Floors, Ceilings).
- Interactive Dashboard.
